🖼️ Images

Online image (URL se):

<img src="url" alt="description" width="300">

Image attributes:
- src = Image ka path/URL (required)
- alt = Alternative text (accessibility ke liye important)
- width = Chaudi (pixels me)
- height = Oonchaayi (pixels me)
- title = Hover karne pe jo text dikhta hai

📦 Div & Span (Containers)

DIV = Block level container (puri width leta hai, neeche naya line)
Group of elements ko ek saath rakhne ke liye
Ye ek div hai. Div block element hai.
Div ke andar div bhi ho sakta hai (nested).
SPAN = Inline container (sirf apne content ki width leta, same line me rahta)
Specific text ko target karne ke liye

🎯 Semantic HTML (Meaningful Tags)

Semantic tags ka matlab hai ki tag ka naam hi batata hai ki uske andar kya hai.
SEO (Search Engine Optimization) aur accessibility ke liye important hai.

Article Title (header tag)

Common semantic tags:
- <header> = Page/section ka header
- <nav> = Navigation links
- <main> = Main content
- <article> = Independent content (blog post, news article)
- <section> = Related content ka group
- <aside> = Side content (sidebar)
- <footer> = Page/section ka footer
