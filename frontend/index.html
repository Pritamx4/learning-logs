🔥 Most Important Concept

addEventListener()

Event Listeners = JavaScript ki sabse powerful feature!

User actions (click, hover, type) ko handle karna hi asli interactivity hai.

// Syntax: element.addEventListener('event', function); // Example: button.addEventListener('click', function() { console.log("Button clicked!"); alert("Hello!"); }); // Common events: - click, dblclick - mouseenter, mouseleave - input, change - keydown, keyup - submit, focus, blur
