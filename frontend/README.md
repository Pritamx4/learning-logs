## 🎯 Event Listeners - The Most Important Concept!

JavaScript me sabse important:

```javascript
// Syntax:
element.addEventListener('event', function);

// Example:
button.addEventListener('click', function() {
    console.log("Button clicked!");
});
```

### Common Events:
- `click` - Mouse click
- `input` - Typing in input field
- `change` - Dropdown/checkbox change
- `mouseenter` - Mouse hover start
- `mouseleave` - Mouse hover end
- `keydown` - Keyboard key press
- `submit` - Form submit

**File me 6+ working event listener examples hai!**

## 🐛 Common Errors & Solutions

### HTML Not Showing?
✅ File extension `.html` hai?
✅ Browser me refresh karo (Ctrl+R)
✅ File save kiya?

### CSS Not Working?
✅ Selector sahi hai?
✅ Syntax sahi hai? (semicolon `;` lagaya?)
✅ DevTools me check karo (F12)

### JavaScript Not Working?
✅ Console me error hai? (F12 → Console)
✅ Element ka ID/Class sahi hai?
✅ Script tag `</body>` se pehle hai?
✅ Function call kiya?

## 📞 How to Debug?

### 1. Console.log() Everywhere!
```javascript
console.log("Starting...");
let x = 5;
console.log("x =", x);
```

### 2. Check Console for Errors
```
Press F12 → Console tab
Red errors dikhenge

### 3. Check Element Exists
```javascript
let element = document.getElementById('myId');
console.log(element);  // null nahi hona chahiye
```

### 4. Check Event Attached?
```javascript
button.addEventListener('click', function() {
    console.log("Event working!");  // Click pe ye dikhe
});
```
